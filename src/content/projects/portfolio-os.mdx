## Overview

Portfolio OS is not just a portfolio—it's a living system that represents how I think and build. It combines structured content, knowledge relationships, and AI capabilities into a unified experience.

## System Design

The architecture follows a modular pattern:

- **Content Layer**: MDX-based project documentation
- **Graph Layer**: D3.js visualization of project relationships
- **AI Layer**: OpenAI embeddings for semantic search and Q&A
- **Memory Layer**: Persistent insights that improve over time

### Key Technical Decisions

1. **MDX for Content**: Allows writing with React components inline
2. **Client-side Graph**: D3.js with force simulation for dynamic layout
3. **Streaming Responses**: ReadableStream for real-time AI output
4. **Embeddings Cache**: Pre-computed vectors for fast similarity search

## Challenges

Building a cohesive system while maintaining performance was the primary challenge. The graph visualization required careful optimization to handle large datasets smoothly.

## Outcome

A system that demonstrates full-stack capabilities, AI integration, and systems thinking—exactly the kind of platform I enjoy building.
